# Redux Introduction

Link: [Getting Started with Redux](https://redux.js.org/introduction/getting-started)

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running console

```sh
npm run console
```

## Running browser

```sh
npm run start
```
